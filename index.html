GitHub backup setup
1. Create a token
Go to github.com/settings/tokens → Generate new token (classic)
Give it a name like "Road Alert", set expiration, check only the repo scope, click Generate.
2. Copy the token
It starts with ghp_... Copy it once — GitHub won't show it again. Paste in the token field above.
3. Set repo and path
Repo: whatistheplanhotmailcom-netizen/speedcam
Path: data/road-alert.json (or any path, will be created)
4. Tap "Backup now"
Your data file commits to your repo. Tap any time you've added points. Token stays only on this phone.
